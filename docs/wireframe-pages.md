# Páginas Planejadas — NutriAcesso

## Páginas Públicas

### 1. Home (`/`)
- Hero section com apresentação da plataforma
- Chamada para nutricionistas se cadastrarem
- Chamada para usuários encontrarem profissionais
- Breve explicação de como funciona (3 passos)

### 2. Diretório (`/profissionais`)
- Barra de filtros (estado, especialidade, modalidade)
- Grid de cards dos profissionais aprovados
- Paginação ou scroll infinito

### 3. Perfil do Profissional (`/profissionais/:id`)
- Foto, nome, CRN
- Especialidades
- Localização e modalidade
- Valor social
- Descrição pessoal
- Botões de contato (WhatsApp, e-mail)

### 4. Cadastro de Nutricionista (`/cadastro`)
- Formulário com todos os campos do RF01
- Mensagem de confirmação após envio

### 5. Sobre (`/sobre`)
- O que é a plataforma
- Como funciona
- Para quem é

## Páginas Administrativas

### 6. Login Admin (`/admin/login`)

### 7. Painel Admin (`/admin`)
- Lista de cadastros pendentes
- Opções de aprovar / recusar
- Lista de profissionais ativos com opção de editar/remover
