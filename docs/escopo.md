# Escopo do Projeto — NutriAcesso

## Objetivo

Criar uma plataforma web onde nutricionistas possam se cadastrar voluntariamente para oferecer atendimento nutricional a preço social, disponibilizando seus dados de contato para o público.

## O que a plataforma FAZ

- Exibe um diretório de nutricionistas cadastrados
- Permite filtrar profissionais por localidade, especialidade e modalidade
- Exibe o perfil de cada profissional com suas informações de contato
- Permite que nutricionistas se cadastrem via formulário
- Administração aprova os cadastros antes de publicar

## O que a plataforma NÃO faz

- ❌ Não realiza videochamadas
- ❌ Não é plataforma de teleconsulta
- ❌ Não faz agendamento online
- ❌ Não processa pagamentos

## Público-alvo

- **Nutricionistas**: profissionais que desejam oferecer atendimento acessível
- **Usuários**: pessoas que buscam atendimento nutricional a preço social

## Fluxo Principal

### Nutricionista
1. Acessa a página de cadastro
2. Preenche formulário com dados profissionais e de contato
3. Aguarda aprovação do administrador
4. Perfil publicado no diretório

### Usuário
1. Acessa o diretório de profissionais
2. Filtra por localidade / especialidade
3. Visualiza perfil do profissional
4. Entra em contato diretamente pelo meio informado (WhatsApp, e-mail, telefone)
