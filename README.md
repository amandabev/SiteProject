# NutriAcesso 🥗

Plataforma de conexão entre nutricionistas voluntários e pessoas que buscam atendimento nutricional a preço social.

## Sobre o Projeto

O NutriAcesso é um diretório online onde nutricionistas podem se cadastrar para oferecer atendimento a preço social. A plataforma centraliza os contatos dos profissionais — **não realiza videochamadas ou teleconsulta diretamente**.

## Funcionalidades Planejadas

- [ ] Cadastro de nutricionistas (nome, CRN, especialidade, formas de contato)
- [ ] Listagem pública de profissionais disponíveis
- [ ] Filtros por estado, especialidade e modalidade de atendimento
- [ ] Página de perfil do profissional com informações e contato
- [ ] Painel administrativo para aprovação de cadastros
